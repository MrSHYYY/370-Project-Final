* One Scoreboard

One Scoreboard is a web-based sports score management system built with PHP and MySQL. It allows users to record match scores, track team and player performance, schedule upcoming matches, view past results, and compare teams through head-to-head records.

* Features
1.User registration and login
2.Admin login and dashboard
3.Sports selection window
4.Match score entry system
5.Team-based score entry
6.Individual player performance tracking from dashboard
7.Football score support with goals, cards, and ratings
8.Cricket score support with runs, wickets, overs, and extras
9.Scheduled match creation
10.Completed match history
11.Past score viewing
12.Head-to-head team comparison with past scores
13.Admin control for deleting users and match records
14.Real-time score validation warnings

* Tech Stack :
PHP
MySQL
HTML
CSS
XAMPP / Apache Server
